# Card
Наследует [Div Props](/docs/en/Div).

Карточки содержат контент и действия, относящиеся к одной теме.

## Базовый пример

```jsx
import { Card, Span, Br, H4 } from '@startupjs/ui'
```

```jsx example
return (
  <Card onPress={() => console.log('You clicked the card')}>
    <H4>Card header</H4>
    <Br />
    <Span size='xl'>Card body or description</Span>
  </Card>
)
```

## Card Level Example

Card `level` свойство определяет насколько высоко будет она приподнята.
Это имеет смысл когда `variant` - `elevated`.

```jsx
import { Card, Span, Br, H4 } from '@startupjs/ui'
```

```jsx example
return (
  <Div>
    <Card>
      <H4>Card Level - 1</H4>
      <Br />
      <Span size='xl'>Card body or description</Span>
    </Card>
    <Br />
    <Card level={2}>
      <H4>Card Level - 2</H4>
      <Br />
      <Span size='xl'>Card body or description</Span>
    </Card>
    <Br />
    <Card level={3}>
      <H4>Card Level - 3</H4>
      <Br />
      <Span size='xl'>Card body or description</Span>
    </Card>
    <Br />
    <Card level={4}>
      <H4>Card Level - 4</H4>
      <Br />
      <Span size='xl'>Card body or description</Span>
    </Card>
  </Div>
)
```

## Card Variant пример

`variant` может быть `elevated` (по умолчанию) либо `outlined`.

import { Card, Span, Br, H4, Icon } from '@startupjs/ui'
