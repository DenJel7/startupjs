# Br

`Br` is used to insert a line break. It also lets the developer customize the number of line breaks needed.

## Example

Number of line breaks can be customized by passing a number to `lines` prop. Default value is `1`.

```jsx
import { Br, Content, Div, Span } from 'startupjs/ui'
```

```jsx example
const itemStyle = {
  height: u(5),
  justifyContent: 'center',
  alignItems: 'center'
}
const [value, setValue] = useState('1')
return (
  <Content>
    <TextInput
      label='Enter number of lines of space'
      value={value}
      onChangeText={(text) => setValue(text)}
    />
    <Br />
    <Div style={itemStyle} level={1}>
      <Span size='xl'>Item 1</Span>
    </Div>
    <Br lines={Number(value)} />
    <Div style={itemStyle} level={1}>
      <Span size='xl'>Item 2</Span>
    </Div>
  </Content>
)
```

## Half of line break

Setting the `half` prop to `true` creates a line break equivalent to a `lines` value of `0.5`.

```jsx
import { Br, Content, Div, Span } from 'startupjs/ui'
```

```jsx example
const itemStyle = {
  height: u(5),
  justifyContent: 'center',
  alignItems: 'center'
}
return (
  <Content>
    <Div style={itemStyle} level={1}>
      <Span size='xl'>Item 1</Span>
    </Div>
    <Br half />
    <Div style={itemStyle} level={1}>
      <Span size='xl'>Item 2</Span>
    </Div>
  </Content>
)
```
