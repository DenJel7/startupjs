# Breadcrumbs

Breadcrumbs позволяют пользоватлю передвигаться по иерархии родителей и отображают текущее состояние внутри иерархии.

## Базовый пример

Каждый объект маршрута принимает любые свойства компонента [`Link`](/docs/en/docs/Link) кроме тех что упомянуты ниже.

```jsx
import { Breadcrumbs } from '@startupjs/ui'
```

```jsx example
return (
  <Breadcrumbs
    routes={[
      {
        to: '#',
        name: 'StartupJs'
      },
      {
        to: '#',
        name: 'Docs'
      },
      {
        to: '#',
        name: 'Component'
      }
    ]}
  />
)
```

## Пример с кастомным разделителем

Разделитель по умолчанию - `/`. Он может быть задан любым строковым значением используя `separator` свойство.

```jsx
import { Breadcrumbs } from '@startupjs/ui'
```

```jsx example
return (
  <Breadcrumbs
    separator='>'
    routes={[
      {
        to: '#',
        name: 'StartupJs'
      },
      {
        to: '#',
        name: 'Docs'
      },
      {
        to: '#',
        name: 'Component'
      }
    ]}
  />
)
```

## Пример с иконкой

Каждому объекту маршрута можно задать иконку. Расположение иконки задается используя `iconPosition` свойство. По умолчанию `iconPosition` - `left`.

```jsx
import { Breadcrumbs } from '@startupjs/ui'
import { faHouseUser, faTable } from '@fortawesome/free-solid-svg-icons'
```

```jsx example
return (
  <Breadcrumbs
    routes={[
      {
        to: '#',
        name: 'House',
        icon: faHouseUser
      },
      {
        to: '#',
        name: 'Table',
        icon: faTable,
        iconPosition: 'right'
      }
    ]}
  />
)
```

## Пример с заданным размером

Размер по умолчанию - `m`, свойство `size` используется для задания размера. Возможные значения свойства `size` можно найти в разделе `Песочница` внизу страницы.

```jsx
import { Breadcrumbs } from '@startupjs/ui'
```

```jsx example
return (
  <View>
    <Breadcrumbs
      size='l'
      routes={[
        {
          to: '#',
          name: 'StartupJs'
        },
        {
          to: '#',
          name: 'Docs'
        },
        {
          to: '#',
          name: 'Component'
        }
      ]}
    />
    <Br />
    <Breadcrumbs
      size='s'
      routes={[
        {
          to: '#',
          name: 'StartupJs'
        },
        {
          to: '#',
          name: 'Docs'
        },
        {
          to: '#',
          name: 'Component'
        }
      ]}
    />
  </View>
)
```
