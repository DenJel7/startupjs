# Tag
Inherits [Button Props](/docs/en/Button).

Tags are compact elements that represent an input, attribute, or action. They allow users to enter information, make selections, filter content, or trigger actions. While buttons are expected to appear consistently and with familiar calls to action, tags should appear dynamically as a group of multiple interactive elements.

## Examples

### Variant

`variant` prop can be either `flat` (default) or `outlined`.

```jsx
import { Tag, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Tag>TAG-1</Tag>
    <Tag variant='outlined' pushed>TAG-1</Tag>
    <Tag color='error' pushed>TAG-2</Tag>
    <Tag color='error' variant='outlined' pushed>TAG-2</Tag>
    <Tag color='#6F00AC' pushed>TAG-3</Tag>
    <Tag color='#6F00AC' variant='outlined' pushed>TAG-3</Tag>
  </Row>
)
```

### Shape

`shape` prop can be either `circle` (default) or `rounded`.

```jsx
import { Tag, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Tag>CIRCLE</Tag>
    <Tag shape='rounded' pushed>ROUNDED</Tag>
  </Row>
)
```

### Icon

Custom icon can be included using `icon` prop, `iconPosition` can be either `left` (default) or `right`.

Tag can be made interactive using `onPress` prop.

```jsx
import { Tag, Row } from '@startupjs/ui'
```

```jsx example
const [showTags, setShowTags] = useState([true, true])
return (
  <Row>
    <Tag
      color='blue'
      style={{ marginRight: u(1) }}
      icon={faCoffee}
    >
       Non Interactive
    </Tag>
    {showTags[0] &&
      <Tag
        color='blue'
        style={{ marginRight: u(1) }}
        icon={faTimesCircle}
        onPress={() => setShowTags([false, showTags[1]])}
      >
        Interactive Left
      </Tag>
    }
    {showTags[1] &&
      <Tag
        color='blue'
        style={{ marginRight: u(1) }}
        icon={faTimesCircle}
        iconPosition='right'
        onPress={() => setShowTags([showTags[0], false])}
      >
        Interactive Right
      </Tag>
    }
  </Row>
)
```
